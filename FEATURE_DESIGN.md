## [001] 海外 AI 订阅教程页面 ✅ 代码已推送，🚀 待部署

### 背景
用户首次订阅 Claude / ChatGPT / Gemini 等海外 AI 工具时缺乏引导，容易踩坑（绑卡失败、账号被封等）。教程页放在 VPN 旁边，形成「VPN + 卡 + 教程」的完整闭环。

### 页面路由
`/guide` → `app/guide/page.tsx`（纯前端，无需新 API）

### 入口位置
1. **Dashboard 快捷操作区** — VPN 卡片旁新增「📖 订阅教程」卡片
2. **Header 下拉菜单** — "临时 VPN" 下方增加 "订阅教程" 链接
3. **Header 移动端菜单** — 同上

### 页面结构
```
左侧固定目录（7 个章节）| 右侧内容区
```

| 章节 | 内容 |
|------|------|
| 🧰 订阅前必读 | 需要准备什么、虚拟卡限制、账号安全 |
| 🤖 Claude | 注册 + Pro 订阅 + API 配合编程工具 |
| 💬 ChatGPT | 注册（含手机号接码）+ Plus 订阅 + Team/API |
| ✨ Gemini | Gmail 直接登录 + Google One AI Premium |
| 🦾 Grok | 注册 + SuperGrok + xAI API |
| 🖥️ Cursor | 下载 + Pro 订阅 + 配置本平台 API |
| 💳 虚拟卡使用技巧 | 充值建议 + 防续费失败 + 常见支付失败排查 |

### 交互设计
- 左侧目录点击切换章节
- 每章节内的步骤可折叠展开（默认全展开）
- 底部有「上一章 / 下一章」导航
- 最后一章末尾有「去开卡订阅」按钮跳回 Dashboard

### 注意事项提示样式
- ⚠️ 红色警告框（关键风险）
- ✓ 绿色列表（小技巧 / 注意事项）
